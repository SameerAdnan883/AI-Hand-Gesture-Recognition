# AI Hand Gesture Recognition

Real-Time Hand Gesture Recognition System built using Python, OpenCV, and MediaPipe.

## Project Overview

This project uses MediaPipe Hand Landmarker to detect 21 hand landmarks in real time through a webcam and recognize predefined hand gestures based on finger positions.

## Features

* ✊ FIST Detection
* ✋ OPEN PALM as STOP Detection
* ✌️ VICTORY Detection
* ☝️ POINTING Detection
* 🤟 LOVE YOU Detection
* 🤘 ROCK Detection
* Real-Time Webcam Processing
* Hand Landmark Visualization

## Tech Stack

* Python



## Run the Project

```bash
python gesture_detection_cam_4.py
```

## Supported Gestures

| Gesture      | Description                   |
| ------------ | ----------------------------- |
| ✊ FIST       | All fingers closed            |
| ✋ OPEN PALM  | All fingers open              |
| 👍 THUMBS UP | Thumb open                    |
| ✌️ PEACE     | Index and middle fingers open |
| ☝️ POINTING  | Index finger open             |
| 🤟 LOVE YOU  | Thumb, index, and pinky open  |
| 🤘 ROCK      | Index and pinky open          |
